# File Organizer

A lightweight utility for organizing files into folders.

The application scans a directory, classifies files by type and prepares a moving plan.

---

## Preview

```
Scanning Downloads...

Documents
    report.pdf
    notes.docx

Images
    photo.jpg
    logo.png

Archives
    backup.zip

Videos
    movie.mp4

Processed files : 6
Folders created : 4
```

---

## Folder Categories

| Category | Extensions |
|----------|------------|
| Images | jpg, png, gif |
| Documents | pdf, docx, txt |
| Archives | zip, rar |
| Videos | mp4, avi |

---

Run

```
python app.py
```

This demo works with sample data only.
